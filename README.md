# 지진 대피소 찾기
- 지진 대피소 공공 데이터 기반 지진 대피소 검색 사이트
- 컴퓨터공학과 3학년 2학기 데이터베이스 프로그래밍 과제로 수행한 프로젝트

## 기능
- 시설명, 도로명주소, 지번주소로 지진 대피소를 검색할수있음
- 검색 결과들 중에서 대피소를 선택해서 대피소의 위치, 수용인원수 등의 정보 확인
- 위치는 카카오맵 API로 보여줌
- 연도 별 지진 발생 통계 자료를 시각과하여 보여줌
